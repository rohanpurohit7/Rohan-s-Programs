## Card Template

Title: Polish repository name

Checklist:
- Preserve original documents
- Improve README structure
- Add or update notebook runbook
- Add command and screenshot placeholders or evidence
- Add validation checklist
- Add compatibility notes
- Add current platform equivalent
- Confirm public-readiness review

## Manual GitHub UI Steps

1. Open GitHub Projects.
2. Create a new table or board project.
3. Add the fields above.
4. Create saved views using the suggested filters.
5. Add repository cards by area.
6. Use Portfolio Ready for pinned or recruiter-facing repositories.
