## How to Use This Portfolio

1. Start with repositories that match the role area.
2. Open each README first for the project story.
3. For lab repositories, check the `notebooks/` folder for runbook-style guides.
4. Use future-state sections to separate original lab work from modernization ideas.
5. Treat original uploaded documents as preserved source artifacts.
